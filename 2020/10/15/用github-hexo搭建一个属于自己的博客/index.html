用github+hexo搭建一个属于自己的博客

无标签
发布日期:   2020-10-15

前言：

一直想搭建一个自己的博客，但是一直不知道怎么搞，在看别人文章是偶然发现了别人的博客，心想自己也搭建一个吧，于是有了这篇文章 (●’◡’●)。
本篇文章参考 https://422926799.github.io/posts/5358472d.html

我的环境: win10

正文：

首先需要一个Github账号

Github官网：https://github.com

注册好之后，去新建一个项目。项目名称为你的username.github.io 我也不知道为什么必须要这样，可能害怕有重名的问题吧。

然后需要的环境是git，nodejs，hexo，npm

  1. git安装直接官网下载你系统版本 https://git-scm.com/download/win
  2. Linux下安装： sudo apt-get install git
  3. windows安装nodejs：https://www.cnblogs.com/hackyo/p/8110951.html //安装nodejs教程和npm教程
  4. Linux下安装nodejs：https://blog.csdn.net/u014361775/article/details/78865582
  5. 安装hexo: 打开命令行输入 npm i -g hexo 如果没反应等情况,可以看看本篇文章https://blog.csdn.net/weixin_38802323/article/details/102586628?utm_medium=distribute.pc_relevant_t0.none-task-blog-BlogCommendFromMachineLearnPai2-1.channel_param&depth_1-utm_source=distribute.pc_relevant_t0.none-task-blog-BlogCommendFromMachineLearnPai2-1.channel_param
  6. 查看hexo是否安装成功:hexo -v
  7. 这时我们创建一个文件夹 例如：blog_hexo 我们命令行切换进去
  8. 输入hexo init进行初始化，成功后blog_hexo中会出现许多文件(注意hexo命令最好在blog_hexo文件夹中输入)
  9. 接下来我们就可以通过命令来搭建本地的博客 hexo generate
  10. hexo server //启动服务
    打开浏览器访问命令行中的地址就可以看到我们的博客了

如果想要更换主题可以去官网下载，我这里用的是https://github.com/blinkfox/hexo-theme-matery

我们打开git Bash 并切换目录到themes里 输入：git clone https://github.com/blinkfox/hexo-theme-matery.git
下载喜欢的主题。

下载成功后将文件夹的名字复制，打开根目录下的_config.yml文件，用notepad或其他编辑器打开，找到theme 将内容修改为我们复制的内容，然后输入命令：

hexo clean

hexo generate

hexo server

打开命令行中的网站就可以查看我们修改过的博客。

做好这些后我们需要将我们的博客迁移到github上让更多人看见，就像我现在这样 (●’◡’●)

弄好之后我们执行hexo d -g

浏览器访问：https://你的username.github.io/ 如果找不到，我们可以找到我们github上面的项目 点击settings一直往下拉，找到GitHub Pages

Your site is published at https://daveqqq.github.io/

这个链接就是你的博客地址哦~

接下来我们开始写文章
hexo new “filename”
输入命令后在 source/_post 中就会出现我们的文件是以md后缀结尾的文件。
我们需要用Markdown来编写我们的页面。

我这里使用的vscode来编写的具体的配置过程 https://zhuanlan.zhihu.com/p/56943330

还有对于我们主题的配置修改请移步你所下主题的具体配置文件，这里就不多说了。
